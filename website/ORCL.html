ORCL

Oracle is a multinational technology company headquartered in Redwood City, CA. Oracle was founded by Larry Ellison, Ed Oates, and Bob Miner in 1977. Oracle was famous for developing one of the first relational database software systems. However, as time has progressed Oracle has expanded its product line to include cloud computing and application support. Oracle is currently traded on the New York Stock Exchange (NYSE).
